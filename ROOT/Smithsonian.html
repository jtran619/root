SMITHSONIAN

A fun and modern update to the Smithsonian Institution’s more traditional and outdated website. The new page is a single scroll website built from HTML, CSS, and JavaScript. The overachring stylistic theme is an organic stroke used across the webpage. It is seen replacing the “O” in Smithsonian, used as an indicator when hovering over the navigation bar, and as arrows.
